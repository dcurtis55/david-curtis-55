# David Curtis — Portfolio

A responsive personal portfolio for David Curtis, built with semantic HTML, modern CSS, and lightweight JavaScript. The site presents David's education, work experience, leadership, community service, and contact information in a polished single-page experience.

## View locally

Open `index.html` directly, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Publish with GitHub Pages

In the repository's **Settings → Pages**, choose **Deploy from a branch**, select the current branch and `/ (root)`, then save. The site uses relative asset paths and requires no build step.
